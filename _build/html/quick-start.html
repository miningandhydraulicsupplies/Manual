Quick Start Guide

Welcome to MHS WMS

This quick start guide will help you get up and running with the MHS Warehouse Management System in minutes.

First Login

Accessing the System

  1. Open your web browser (Chrome, Firefox, or Edge recommended)

  2. Navigate to: https://warehouse.mhs.com.au

  3. You’ll see the login page

Logging In

  1. Enter your email address as username

  2. Enter your password

  3. Click Login

  4. You’ll be redirected to the Dashboard

First-Time Login

If this is your first login:

  1. You may be prompted to change your password

  2. Choose a strong password meeting requirements:

    • At least 6 characters

    • One uppercase letter

    • One lowercase letter

    • One number

    • One special character

  3. Click Change Password

Dashboard Overview

After logging in, you’ll see the main dashboard with:

Key Sections

Daily Statistics (Top)

  • Today’s label printing activity

  • Quick metrics at a glance

System Status (Right sidebar)

  • Health of connected systems

  • API connectivity status

Recent Activity (Center)

  • Latest operations and print jobs

  • Real-time updates

Quick Actions (Various locations)

  • Jump to common tasks

  • Access frequently used features

Common Tasks

Printing a Pickslip Label

  1. Click Labels in sidebar

  2. Select Pickslip

  3. Enter pickslip number

  4. Select printer

  5. Click Print

Time to complete: 30 seconds

Adding Item to Order Board

  1. Click Order Board in sidebar

  2. Select division tab (Hydraulics, Seals, etc.)

  3. Click Add Item

  4. Enter part number and quantity

  5. Click Save

Time to complete: 1 minute

Checking Order Status

  1. Click Order Monitor in sidebar

  2. Use search box to find order

  3. Click order to view details

  4. Review status and line items

Time to complete: 30 seconds

Recording a Calibration

  1. Click Calibration Tracking in sidebar

  2. Search for equipment

  3. Click Record Calibration

  4. Enter calibration details

  5. Upload certificate (optional)

  6. Click Save

Time to complete: 2 minutes

Keyboard Shortcuts

Learn these shortcuts to work faster:

  • Ctrl + F - Focus search box

  • Ctrl + P - Quick print (on label pages)

  • Ctrl + Enter - Submit form

  • Esc - Close dialog

  • F5 - Refresh page

Troubleshooting

Common Issues

Can’t Login:

  • Verify email and password

  • Check Caps Lock is off

  • Try password reset

  • Contact admin if locked out

Page Won’t Load:

  • Check internet connection

  • Try refreshing (F5)

  • Clear browser cache

  • Try different browser

Printer Not Working:

  • Check printer is online

  • Verify printer selection

  • Check print queue

  • Contact IT if issue persists

Data Not Updating:

  • Refresh the page

  • Check for error messages

  • Wait for next sync cycle

  • Report to admin if persistent
